# Spring MVC Bookstore

A mini project built with Spring MVC, Thymeleaf, Spring Data JPA, MySQL.

## Features
-> Add new books
-> Edit books
-> Delete books
->View book list

## Technologies
-> Spring Boot
-> Spring MVC
-> Thymeleaf
-> Spring Data JPA
-> MySQL
